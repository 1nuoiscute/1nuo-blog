# 电路分析 Lesson 21：交流电路功率与谐振

> 本课继续讨论交流电路的功率问题，涵盖电路性质判断、功率因数的提高、最大功率传输条件，以及串联与并联谐振的基本概念。重点掌握复功率的计算方法、功率因数补偿的工程意义，以及谐振条件与特征。

## 一、交流电路的功率与性质判断

### 1.1 电压电流相位关系与电路性质

根据电压与电流的相位关系判断电路性质：
- 电压超前于电流 → 电路呈**感性**（类似于电感）
- 电流超前于电压 → 电路呈**容性**
- 电压与电流同相 → 电路呈**纯阻性**

阻抗角 $\varphi$ 可由 $\arccos$ 或 $\arctan$ 求出，其正负取决于电路偏感性还是偏容性。

### 1.2 极端情况分析

| 情况 | 条件 | 阻抗角 $\varphi$ | 特征 |
|------|------|-------------------|------|
| 纯电阻 | $X = 0$，$Z = R$ | $\varphi = 0^\circ$ | 仅消耗有功功率，$\cos\varphi = 1$ |
| 纯电抗 | $R = 0$，$Z = jX$ | $\varphi = \pm 90^\circ$ | 无有功消耗，仅电感和电容间能量交换 |
| 感性 | $X > 0$ | $\varphi > 0$ | 电流滞后电压，$\cos\varphi$ 称为**滞后** |
| 容性 | $X < 0$ | $\varphi < 0$ | 电流超前电压，$\cos\varphi$ 称为**超前** |

## 二、功率计算

### 2.1 复功率与有功、无功功率

复功率定义为：
$$
\mathbf{S} = \mathbf{U} \cdot \overline{\mathbf{I}} = P + jQ
$$

- 有功功率：$P = UI\cos\varphi = I^2R$
- 无功功率：$Q = UI\sin\varphi = I^2X$
- 视在功率：$S = UI$
- 功率因数：$\cos\varphi = \dfrac{P}{S}$

### 2.2 端口网络功率的多种计算方法

**例题：** 已知各支路电流，求端口网络吸收的功率。

方法一（元件求和法）：分别求各支路电流 $\dot{I}_1$、$\dot{I}_2$，各元件独立计算功率后求和。

方法二（复功率法）：先求总电流 $\dot{I} = \dot{I}_1 + \dot{I}_2$，再用端口电压和总电流求复功率。

方法三（直接公式法）：用 $P = UI\cos\varphi$ 和 $Q = UI\sin\varphi$ 分别计算。

> 多种计算方法可互相验证，是解题中减少错误的有效手段。

### 2.3 并联负载的总功率

已知各支路的 $P$ 和 $\cos\varphi$，求总功率：
- 总有功直接相加：$P = P_1 + P_2$
- 总无功代数和：$Q = Q_1 + Q_2$（感性正、容性负）
- 总视在功率：$S = \sqrt{P^2 + Q^2}$

## 三、功率因数的提高

### 3.1 提高原因

1. **充分利用供电设备容量**：$P = S\cos\varphi$，$\cos\varphi$ 越小需要的 $S$ 越大
2. **减少线路损失**：$P$ 一定时 $\cos\varphi$ 越大，$I$ 越小，线路损耗 $I^2R$ 越小

### 3.2 方法：感性负载并联电容

**为何并联而非串联？** 串联会改变负载电压，影响正常工作。

电容大小计算公式：
$$
C = \frac{P}{\omega U^2} (\tan\varphi_1 - \tan\varphi_2)
$$

### 3.3 欠补偿原则

电力系统采用**欠补偿**（补偿后仍略带感性）：
1. 过补偿需更大电容，不经济
2. 完全补偿可能在特定频率下诱发**谐振**

---

## 四、最大传输功率

含源一端口等效为戴维南电路后，负载 $Z_L$ 获得最大功率的条件（**共轭匹配**）：

$$
Z_L = \overline{Z_{\text{eq}}}
$$ 
即 $R_L = R_{\text{eq}}$，$X_L = -X_{\text{eq}}$

最大功率：
$$
P_{\text{max}} = \frac{U_{\text{oc}}^2}{4R_{\text{eq}}}
$$

## 五、电路的谐振

### 5.1 定义

含有 $RLC$ 的一端口电路，在特定条件下出现**端口电压与电流同相位**的现象，称为谐振。

### 5.2 串联谐振

| 项目 | 内容 |
|------|------|
| 条件 | $X_L = X_C$，即 $\omega L = 1/(\omega C)$ |
| 谐振角频率 | $\omega_0 = 1/\sqrt{LC}$ |
| 阻抗特征 | 最小（$Z=R$），电流最大 |
| 电压特征 | $U_L$ 与 $U_C$ 大小相等、相位相反 |
| 别称 | **电压谐振** |

品质因数：$Q = U_L/U = U_C/U = \omega_0 L/R$

### 5.3 并联谐振

| 项目 | 内容 |
|------|------|
| 条件 | $B_L = B_C$（电感电纳 = 电容电纳） |
| 谐振角频率 | $\omega_0 = 1/\sqrt{LC}$（与串联相同） |
| 阻抗特征 | 导纳最小，阻抗最大 |
| 电流特征 | 总电流最小，支路电流可能远大于总电流 |
| 别称 | **电流谐振** |

### 5.4 应用与危害

- **应用**：收音机调频（串联谐振选择特定频率信号）
- **危害**：电力系统谐振导致过电压/过电流，损坏设备

## 六、关键总结

| 知识点 | 要点 |
|--------|------|
| 复功率 | $\mathbf{S} = \mathbf{U} \cdot \overline{\mathbf{I}} = P + jQ$ |
| 有功功率 | $P = UI\cos\varphi = I^2R$ |
| 无功功率 | $Q = UI\sin\varphi = I^2X$ |
| 功率因数 | $\cos\varphi = P/S$，滞后（感性）或超前（容性） |
| 功率因数提高 | 感性负载并联电容，$C = P/(\omega U^2)(\tan\varphi_1 - \tan\varphi_2)$ |
| 最大功率传输 | $Z_L = \overline{Z_{\text{eq}}}$（共轭匹配） |
| 串联谐振 | $X_L = X_C$，$Z$ 最小 $I$ 最大，电压谐振 |
| 并联谐振 | $B_L = B_C$，$Z$ 最大 $I$ 最小，电流谐振 |
| 谐振频率 | $\omega_0 = 1/\sqrt{LC}$ |
