# 电路分析 Lesson 10：回路电流法应用实例

> **课程内容**：含多个独立电流源的电路运用回路电流法求解，重点练习选树策略与列方程。

---

## 一、为什么用回路电流法而不是网孔法？

| 情况 | 推荐方法 |
|:---|:---|
| 常规电路（无电流源） | 网孔电流法（更简单，不用画树） |
| 含 1 个独立电流源 | 网孔法/回路法都可以（超网孔或选树） |
| **含多个独立电流源** | **回路电流法 + 选树策略**（更系统） |
| 含受控源 | 回路法 + 补控制方程 |

---

## 二、典型例题：含两个独立电流源

### 例题描述

电路中有两个独立电流源（可能包含受控源），要求求某条支路的电流 i₁。

### 解题步骤

**1. 确定节点数，选树**

- 电路有 5 个节点
- 树枝数 = n − 1 = 4
- 策略：**电流源选为连支**
  - 两个电流源所在的支路**不能当树枝**
  - 从剩余支路中选出 4 条构成一棵树
  - 树必须满足：连通、含全部节点、无回路

**2. 老师选的树**

四条树枝构成树，两个电流源支路作为连支：
- 树支路从外围走，避开电流源
- 确保覆盖所有 5 个节点
- 没有形成闭合回路

**3. 确定基本回路**

每条连支 + 若干树枝 = 一个基本回路
- 连支① → 基本回路 1（回路电流 = 电流源电流，已知）
- 连支② → 基本回路 2（回路电流 = 电流源电流，已知）
- 剩余连支 → 基本回路 3、4...（需列 KVL 方程求解）

**4. 列方程**

- 电流源所在的回路 → **不需要列方程**（电流已知）
- 其他基本回路 → 列 KVL 方程
- 如果还有**受控源** → 再补充控制变量的方程

**5. 求解**

联立方程求得回路电流 → 由回路电流求支路电流 i₁

---

## 三、选树的练习技巧

| 节点数 | 树枝数 | 策略 |
|:---|:---|:---|
| 3 | 2 | 避开电流源，选两条不形成回路的支路 |
| 4 | 3 | 像一棵树一样发散出去 |
| 5 | 4 | 从外围绕，或从中间发散 |
| n | n−1 | 树连接所有节点且无回路 |

**要点**：
1. 先标出所有**节点**
2. 把电流源支路**标记出来**（选为连支）
3. 从剩下的支路中选 n−1 条构成一棵树
4. 检查：树是否连通？是否覆盖所有节点？是否有回路？

> 💡 选树不唯一！做作业时发现和答案选的树不同很正常，结果应该一样。

---

## 四、回路电流法完整流程总结

```
1. 确定节点数 n，总支路数 b
2. 选树（避开电流源选连支，避开电压源选树枝）
3. 树枝数 = n−1，连支数 = b−(n−1)
4. 每条连支对应一个基本回路
5. 电流源所在回路 → 电流已知，不列方程
6. 其他回路 → 列 KVL 方程
7. 有受控源 → 补控制方程
8. 求解 → 由回路电流求支路电流
```

---

## 💡 关键总结

| 要点 | 说明 |
|:---|:---|
| 多个电流源 | 回路法比网孔法更方便，选树即可系统处理 |
| 电流源→连支 | 回路电流已知，少列方程 |
| 选树不唯一 | 不同树→不同回路→方程不同→结果相同 |
| 无电流源 | 直接网孔法更快，不用画树 |
| 有受控源 | 当独立源处理，再补控制方程 |
