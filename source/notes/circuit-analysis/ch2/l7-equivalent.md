# 电路分析 Lesson 7：电阻与电源的等效变换

> **课程内容**：电阻等效变换的三种主要方法（串并联简化、惠斯通电桥、星-三角变换），实际电源的等效变换，对外等效原则，以及多余元件的处理。

---

## 一、电阻等效变换的三种基本方法

| 方法 | 适用场景 |
|:---|:---|
| 串并联简化 | 结构清晰的简单电路 |
| 惠斯通电桥平衡条件 | 四边形结构中间有支路的电路 |
| 星-三角（Y-Δ）变换 | 无法直接识别串并联的复杂电阻网络 |

---

## 二、串并联简化

### 1. 基本思路

从端口看进去，由远及近一步步合并电阻。

**例题**：从 ab 端口看进去的等效电阻
- 从最远端开始合并：先看并联 → 串联 → 再并联 → 再串联
- 注意开路支路（电流为 0 的电阻可直接忽略）

### 2. 找节点法

当电阻之间的串并联关系不好判断时：

1. **先标节点**，把电路中所有独立节点标记出来
2. 电位相等的点可合并为同一节点
3. 重新画出两两节点之间的电阻连接关系
4. 原本复杂的电路就变成了直观的串并联结构

> 💡 **关键**：等电位点之间就算隔了一根导线，电位也是相等的，可视为同一个节点。

## 三、惠斯通电桥平衡条件

### 1. 电桥结构

- 四边形 ABCD，四个桥臂电阻 R₁、R₂、R₃、R₄
- 中间一条支路（检测电流 i₅ 或电压表）
- 对外两个端口（输入/输出）

### 2. 平衡条件

**R₁ × R₄ = R₂ × R₃**

即**对边电阻乘积相等**。

### 3. 平衡时的处理

当上述条件满足时：
- 中间支路电流为 **0**
- 可将中间支路视为**开路** 或**短路**处理
- 电路简化为纯串并联结构

### 4. 使用步骤

1. 看到四边形 + 中间支路的结构
2. 检查四个桥臂电阻是否满足对边乘积相等
3. 若满足 → 去掉中间支路 → 电路简化为串并联
4. 若不满则 → 考虑星-三角变换

### 5. 典型判断

- 去掉的支路应与端口**垂直**（跨在端口上的电阻不能去掉）

## 四、星形（Y）与三角形（Δ）等效变换

### 1. 为什么需要？

实际电力系统中，三相负载可接成 Y 形或 Δ 形。在电路分析中，当电阻网络既不是串并联也不是电桥时，通过 Y-Δ 变换可转化为串并联结构。

### 2. 核心原则：对外等效

> 变换前后，三个端口的**电压**和**电流**保持不变。
>
> ⚠️ 对内不等效！变换后内部电阻的电流分布不同，但对外部电路完全等效。

### 3. Δ → Y 变换（三角形→星形）

$$R_1 = \frac{R_{12} \times R_{31}}{R_{12} + R_{23} + R_{31}}$$

$$R_2 = \frac{R_{12} \times R_{23}}{R_{12} + R_{23} + R_{31}}$$

$$R_3 = \frac{R_{23} \times R_{31}}{R_{12} + R_{23} + R_{31}}$$

口诀：**分母是三电阻之和，分子是相邻两电阻乘积**

### 4. Y → Δ 变换（星形→三角形）

$$R_{12} = \frac{R_1 R_2 + R_2 R_3 + R_3 R_1}{R_3}$$

$$R_{23} = \frac{R_1 R_2 + R_2 R_3 + R_3 R_1}{R_1}$$

$$R_{31} = \frac{R_1 R_2 + R_2 R_3 + R_3 R_1}{R_2}$$

口诀：**分子是两两电阻乘积之和，分母是不相邻的那个电阻**

### 5. 特殊情况：三电阻相等

当三个电阻值相等时：

- Y → Δ：**R_Δ = 3 × R_Y**（外大里小，三角形是星形的 3 倍）
- Δ → Y：**R_Y = R_Δ / 3**

### 6. 变换操作步骤

1. 首先找到三角形的**三个节点**（对外部电路不变的节点）
2. 将三角形内部的三电阻变换为 Y 形（多出一个中心节点 O）
3. 或反之，将 Y 形变换为 Δ 形
4. **守住三个对外节点**不动，就不会画错

## 五、电源的等效变换

### 1. 理想电压源与理想电流源

**电压源串联**：可合并为一个电压源（代数和）
**电压源并联**：必须电压相等，否则不允许并联

**电流源并联**：可合并为一个电流源（代数和）
**电流源串联**：必须电流相等，否则不允许串联

### 2. 实际电压源 ↔ 实际电流源

**变换条件**：
- 内阻相等（Rₛ₁ = Rₛ₂）
- 端口电压-电流关系一致
- 变换前后开路电压和短路电流分别相等

**变换公式**：

$$U_s = I_s \times R_s \quad \text{或} \quad I_s = \frac{U_s}{R_s}$$

**方向规则**：
- 电压源 → 电流源时，电流方向由 **电压源负极指向正极**
- 两个电源的参考方向为**非关联**方向

### 3. 方向记忆法

电压源和电流源对外输出时，端口电压和电流关系不变：
- 电压源正极流出电流 → 电流源电流也向外流出
- 它们的方向**相反**（非关联）

### 4. 验证方法

变换是否正确，可以通过检查：
1. **开路电压**是否相等
2. **短路电流**是否相等

## 六、多余元件

### 1. 定义

- 并联在**理想电压源**两端的元件 → 不改变端口电压 → **多余**
- 串联在**理想电流源**支路上的元件 → 不改变支路电流 → **多余**

### 2. 处理方法

- 求**外部电路**参数时：可暂时移除多余元件
- 求**内部**参数时：需还原回来

### 3. 举例

- 电压源两端并联的电阻 → 对外求电流时不影响分压，可去掉
- 电流源支路上串联的电阻 → 对外求电压时不影响分流，可去掉

> ⚠️ 多余元件的概念建立在**对外等效**的基础上。对内，它会影响电压源/电流源的内部分布。

---

## 七、无限链型网络

对于无限重复的链型电阻网络：

1. 设整个网络的等效电阻为 R_ab
2. 去掉首级 r₁、r₂ 后，剩余部分仍然是 R_ab（无限性）
3. 列出方程：R_ab = r₁ + (r₂ ∥ R_ab)
4. 解出正实数解（纯电阻电路的等效电阻恒为正）
5. 负根舍去

## 八、课堂解题方法总结

| 方法 | 什么时候用 |
|:---|:---|
| 串并联 | 一眼能看清电阻关系的电路 |
| 惠斯通电桥 | 四边形 + 中间支路，检查对边乘积 |
| 星-三角变换 | 上述两种都不行时 |
| 等电位点 | 电路对称时，利用对称性折叠 |
| 电源等效变换 | 电源混联时，统一成同种电源 |
| 多余元件法 | 电压源并了东西 / 电流源串了东西 |

---

## 💡 关键总结

| 知识点 | 要点 |
|:---|:---|
| 对外等效 | 变换只保证外部特性不变，内部不等效 |
| 电桥平衡 | R₁×R₄ = R₂×R₃，中间支路可去掉 |
| Y-Δ 变换 | Δ→Y 分母和，分子乘；Y→Δ 分子和，分母除 |
| 三电阻相等 | R_Δ = 3 × R_Y |
| 电源等效 | U = I × R，方向非关联（负→正） |
| 多余元件 | 对外求参数时暂时移除 |
| 无限网络 | 自相似列方程，取正根 |
