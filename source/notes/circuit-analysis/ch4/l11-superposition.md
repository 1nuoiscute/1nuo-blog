# 电路分析 Lesson 11：叠加定理与替代定理

> **课程内容**：叠加定理的适用条件与使用方法，齐次性（比例性），以及替代定理。

---

## 一、叠加定理

### 1. 定理内容

在线性电路中，**多个独立电源**共同作用产生的响应（电压或电流），等于各独立电源**单独作用**时产生的响应的**代数和**。

### 2. 适用条件

> ⚠️ **必须是线性电路**

- 线性电阻、线性受控源 → 可叠加
- 二极管等非线性元件 → ❌ 不可叠加

### 3. 独立电源单独作用时的处理

| 电源类型 | 不作用时替换为 |
|:---|:---|
| **电压源** | **短路**（导线）|
| **电流源** | **开路**（断开）|
| **受控源** | **保留在电路中**（不能单独作用）|

> 受控源不是独立电源，是受其他量控制的，因此**不能单独作用**，要在所有分电路中保留。

### 4. 叠加定理的使用步骤

1. 画出每个独立电源单独作用的分电路图
2. 在其他电源不作用时：电压源短路，电流源开路
3. **受控源始终保留**
4. 分别计算每个分电路中待求的电压或电流
5. 将各分量**叠加**（代数和），注意方向

### 5. 注意事项

| ✅ 可以叠加 | ❌ 不可以叠加 |
|:---|:---|
| 电压 | **功率** |
| 电流 | |

**功率不能叠加**，因为 $P = UI$，功率与电压/电流是**非线性关系**。

$$P \neq P_1 + P_2$$

计算功率时，应先叠加出总电压和总电流，再用总电压×总电流。

## 二、齐次性（比例性）

### 1. 内容

在线性电路中，若**只有一个独立电源**作用，则响应（电压/电流）与激励（电源）成**正比**。

当激励增大 k 倍时，响应也增大 k 倍。

### 2. 应用示例

例题：开关在三个不同位置的电路

```
位置 1：电流源 Is 单独作用 → 测得 I₁ = 40mA
位置 2：电流源 Is + 电压源 Us₁ 共同作用 → 测得 I₂
位置 3：电流源 Is + 电压源 Us₂ 共同作用 → 求 I₃
```

解题思路：
1. 由位置 1 得电流源分量：$I_{Is} = 40\text{mA}$
2. 由位置 2 得电压源 Us₁ 分量：$I_{Us_1} = I_2 - I_{Is}$
3. 由**齐次性**得 Us₂ 分量：$I_{Us_2} = \frac{Us_2}{Us_1} \times I_{Us_1}$
4. 叠加得位置 3 的结果：$I_3 = I_{Is} + I_{Us_2}$

## 三、替代定理

### 1. 定理内容

在电路中，如果已知某条支路的电压 $U_k$ 或电流 $I_k$，则可以用一个**电压源**（值为 $U_k$）或**电流源**（值为 $I_k$）替代该支路，替代后电路中其他部分的电压和电流**保持不变**。

### 2. 替代方式

| 已知量 | 替代为 |
|:---|:---|
| 支路电压 $U_k$ | 电压源 $U_k$（极性相同） |
| 支路电流 $I_k$ | 电流源 $I_k$（方向相同） |

### 3. 适用范围

- 替代定理对**线性**和**非线性**电路都成立
- 但被替代的支路与电路其余部分之间不能有**耦合关系**（如受控源的控制关系）

## 四、叠加定理的典型考题

### 题型特色

1. 先用叠加定理分电源求分量
2. 常结合**齐次性**（比例性）一起考——同一电流源在不同开关位置产生的响应相同
3. 不同开关位置的测量数据，反推特定条件下的结果
4. 注意：**画分电路图**是得分关键（短路电压源、开路电流源）

### 齐次性应用例题

**题目**：电路中有电流源 $i_s$ 和两个电压源 $u_{s1}$、$u_{s2}$，通过不同开关位置测数据：

| 开关位置 | 作用的电源 | 测得电流 |
|:---:|:---|:---:|
| 1 | $i_s$ 单独 | 40mA |
| 2 | $i_s + u_{s1}$ | 含两部分 |
| 3 | $i_s + u_{s2}$（幅值不同） | ？ |

**解法思路**：
1. 位置 1：$i_s$ 单独 → 得 40mA
2. 位置 2：整体 = $i_s单独 + u_{s1}单独$ → 反推 $u_{s1}$ 单独贡献
3. 位置 3：利用齐次性，$u_{s2}$ 的贡献 = $\frac{u_{s2}}{u_{s1}} \times (u_{s1}$ 单独贡献$)$
4. 三者叠加即得结果

### 解题口诀

> 叠加定理线性用，电源分别独立动；
>
> 电压源→短路，电流源→开路，受控源→不动；
>
> 方向一致再加和，功率不能直接叠。

## 💡 关键总结

| 知识点 | 要点 |
|:---|:---|
| 叠加定理适用 | **线性电路** |
| 电源处理 | 电压源→短路，电流源→开路，受控源→保留 |
| 叠加对象 | 电压✅ 电流✅ **功率❌** |
| 齐次性 | 单电源时，响应与激励成正比 |
| 替代定理 | 已知支路U或I，可用电源替代该支路 |
| 替代适用范围 | 线性/非线性均可，但无耦合关系 |
