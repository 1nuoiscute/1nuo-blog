## 🔄 KVL 定律（基尔霍夫电压定律）

### 核心内容

回路中**各元件电压的代数和为零**。

### 巡行方向法（推荐方法）

> 为什么推荐这个方法？避免混淆"电压升"还是"电压降"的分类。

1. 先设定**巡行方向**（顺时针或逆时针）
2. 巡行方向 = 默认的**电压正方向**
3. 元件电压与巡行方向一致 → 取 **正**
4. 元件电压与巡行方向相反 → 取 **负**

### 重要推论：端口电压与路径无关

无论选择哪条路径计算两点间电压，结果恒定。后续分析中常直接利用此性质求解端口电压。

### 电源参考方向的默认约定

> 电源（电压源、电流源）的参考方向应默认设为**非关联方向**，以便于后续功率判别。

## ⚡ 电阻的等效化简

*  **串联**：$R_{eq} = R_1 + R_2 + \cdots$（分压、限流）
*  **并联**：$G_{eq} = G_1 + G_2 + \cdots$（分流）
*  **两电阻并联高频公式**：$R_{eq} = \frac{R_1 \times R_2}{R_1 + R_2}$
*  **分流公式**：$I_1 = I \times \frac{R_2}{R_1 + R_2}$

## 📌 基尔霍夫定律的适用范围

✅ **适用**：集中参数电路（对元件线性/非线性无要求）
❌ **不适用**：分布参数电路的整体首尾端口分析

## ⚡ 电阻的等效化简与受控源
*  电阻串联:等效电阻等于各电阻之和(Req = R1 + R2 + ...),主要作用是分压与限流  。
*  电阻并联:等效电导等于各电导之和,等效电阻为其倒数,主要作用是分流  。
*  两电阻并联高频公式:Req = (R1 * R2) / (R1 + R2)   。分流公式为 I1 = I * R2 / (R1 + R2)  。
*  受控源电路分析:受控源分为四类(VCVS、CCVS、VCCS、CCCS),在方程中的形式与独立源相同,但需引入控制变量   。若该变量未出现在所列方程中,则需补充相应关系式  。
