## 一、叠加定理复习与齐次性应用

### 1. 叠加定理回顾

| 要点 | 说明 |
|:---|:---|
| 适用范围 | **线性电路** |
| 电源处理 | 电压源不作用→短路，电流源不作用→开路 |
| 受控源 | 不能单独作用，**保留在电路**中 |
| 可叠加量 | 电压✅ 电流✅ |
| 不可叠加量 | **功率❌**（非线性关系） |

### 2. 齐次性（比例性）应用例题

**题目**：电路中有一个电流源 $i_s$ 和两个电压源 $u_{s1}$、$u_{s2}$，通过不同开关位置测量数据：

| 开关位置 | 作用的电源 | 测得电流 |
|:---:|:---|:---:|
| 1 | $i_s$ 单独 | 40mA |
| 2 | $i_s + u_{s1}$ | 由两部分叠加 |
| 3 | $i_s + u_{s2}$（幅值不同） | ？ |

**解法思路**：
1. 位置1 → $i_s$ 单独贡献已知 = 40mA
2. 位置2 → 减去 $i_s$ 的贡献 → 得 $u_{s1}$ 单独贡献
3. 位置3 → 利用**齐次性**：$u_{s2}$ 的贡献 = $\frac{u_{s2}}{u_{s1}} \times (u_{s1}$ 单独贡献$)$
4. 位置3总响应 = $i_s$ 贡献 + $u_{s2}$ 贡献

> 🔑 **启示**：考试常将叠加定理与齐次性结合考查。

## 二、替代定理（Substitution Theorem）

### 1. 定理内容

> 任意一个具有**唯一解**的电路中，若某条支路的电压为 $u_k$、电流为 $i_k$，且该支路与其他支路**无耦合关系**，则该支路可以用一个电压为 $u_k$ 的**电压源**或电流为 $i_k$ 的**电流源**来替代，而不影响电路中其他支路的电压和电流。

### 2. 本质

**替代前后的基尔霍夫电压/电流关系不变。**

- 用电压源 $u_k$ 替代后 → 该支路两端电压不变 → 其他支路电压不变 → 其他支路电流不变
- 用电流源 $i_k$ 替代后 → 该支路电流不变 → 其他支路电流不变 → 其他支路电压不变

### 3. 适用范围

| 特性 | 说明 |
|:---|:---|
| 线性/非线性 | **两者均可**（不同于叠加定理） |
| 必要条件 | 被替代支路必须有**唯一解** |
| 限制条件 | 被替代支路与电路其他部分**无耦合关系** |

### 4. 实际应用

#### 应用一：多余元件支路

- 如果一个支路电流是**恒定**的（如电流源），即使再串联其他元件也不改变该支路电流
- 对外部电路而言，该支路可直接用一个电流源替代

#### 应用二：复杂网络分解

```
       ┌───── u₁ ─────┐
       │              │
    ┌──N₁──┐    ┌──N₂──┐
    │      │    │      │
    └──────┘    └──────┘
       │              │
       └───── u₂ ─────┘
            ┌──N₃──┐
            │      │
            └──────┘
```

一个复杂的三网络连接系统，若已知连接端口电压 $u_1$、$u_2$ 恒定，则：
- N₁ 的端口可用电压源 $u_1$ 替代
- N₂ 可用电压源 $u_2$ 替代
- N₃ 可用电压源 $(u_1 - u_2)$ 替代

→ 复杂网络分解为**三个简单网络**分别求解。

### 5. 替代定理解题示例

#### 例1：已知支路电流求电阻

已知通过 $R_x$ 的电流为 $\frac{2}{10}A$，求 $R_x$。

**解**：
1. 已知该支路电流为 $\frac{2}{10}A$ → 用一个电流为 $\frac{2}{10}A$ 的电流源替代该支路
2. 简化后的电路用节点电压法求端口电压 $u$
3. $R_x = u / (\frac{2}{10}) = u \times 5$

#### 例2：多余元件支路的简化

电路中某支路电流恒定为 3A，与待求的 $I_1$ 无关 → 直接用 3A 电流源替代该支路，电路大大简化。

#### 例3：$u_{ab} = 0$ 的应用

已知 $u_{ab} = 0$ → 该支路电流也为 0 → 相当于**开路** → 断开该支路
→ 这也是一种替代！（用开路替代原支路）

## 三、戴维宁定理（Thevenin's Theorem）⭐

### 1. 定理内容

> 任何一个**线性**有源二端网络，对外部电路而言，总可以用一个**电压源 $u_{oc}$ 串联一个电阻 $R_{eq}$** 来等效替代。

```
     ┌──────────┐                 ┌──── R_eq ────┐
     │ 有源线性 │      ════      │              │
     │ 二端网络 │                 │   U_oc       │
     └──────────┘                 └──────────────┘
        a  b                        a       b
```

### 2. 两个关键参数

| 参数 | 符号 | 含义 | 求法 |
|:---|:---:|:---|:---|
| **开路电压** | $u_{oc}$ | 二端网络**开路**时 a、b 两端的电压 | 用电路分析方法求解 |
| **等效电阻** | $R_{eq}$ | 所有独立电源**置零**后，从端口看进去的等效电阻 | 用电阻等效变换方法求解 |

### 3. 电源置零规则

| 电源类型 | 置零方式 |
|:---|:---|
| **电压源** | → **短路**（$u = 0$） |
| **电流源** | → **开路**（$i = 0$） |
| **受控源** | **保留**（不能置零） |

### 4. 证明思路（叠加定理）

戴维宁定理的证明建立在**叠加定理**的基础上，这也是为什么它要求网络是**线性**的。

### 5. 应用场景

| 适合用戴维宁 | 不适合用戴维宁 |
|:---|:---|
| 只关心**某一支路**的电压/电流 | 需要求解电路内部所有信息 |
| 复杂网络中提取**一个负载** | 有受控源时计算较繁琐 |
| **电力系统**中的等效 | 纯串并联可一步到位的 |

> 💡 **实际工程应用**：
> - 整个电力系统对某个用户来说，就是一个戴维宁等效电源
> - 牵引供电系统对电力机车来说，就是一个电压源+内阻
> - 不关心内部结构，只关心对外特性

### 6. 与诺顿定理的关系

- **诺顿定理**：有源线性二端网络可以用一个**电流源**并联一个电阻等效
- 电压源+串联电阻 ⟷ 电流源+并联电阻，两者可**相互转换**
- 掌握了戴维宁定理就掌握了诺顿定理

### 7. 注意事项

- **方向**：$u_{oc}$ 的方向要与端口电压的参考方向一致
- 戴维宁等效中 $u_{oc}$ 与 $i$ 是**非关联参考方向**（电压源上正下负，电流从正极流出）
- 诺顿等效中方向有所不同，需注意

## 四、补充例题

### 例：已知 $u_{ab} = 0$，求电阻 $R$

- $u_{ab} = 0$ → 支路电流也为 0 → 该支路相当于开路
- 断开该支路后，左侧电路变为简单电阻网络
- 通过串并联化简求等效电阻和端口电压

### 例：2V电压源用多大电阻置换而不影响电路？

- 先求 2V 电压源支路的电流
- $R = u / i = 2V / i$
- 用 $R$ 替代电压源后，外部电路特性不变

## 五、与 L11 的联系

| 定理 | 适用范围 | 特点 |
|:---|:---|:---|
| **叠加定理** | 线性电路 | 多个电源分别作用再叠加 |
| **齐次性** | 线性电路 | 单电源时响应与激励成正比 |
| **替代定理** | 线性/非线性均可 | 已知支路U/I，用电源替代 |
| **戴维宁定理** | 线性电路 | 复杂网络→电压源+电阻 |

> 四个定理层层递进，共同构成电路分析简化的核心工具。

## 六、关键总结

| 知识点 | 要点 |
|:---|:---|
| **替代定理本质** | 替代前后 KVL/KCL 关系不变，即 U/I 不变 |
| **替代前提** | 支路有唯一解、无耦合关系 |
| **戴维宁** | 开路电压 $u_{oc}$ + 等效电阻 $R_{eq}$ |
| **电源置零** | 电压源→短路，电流源→开路，受控源→保留 |
| **诺顿** | 戴维宁可互转，一般不单独考 |
| **考试重点** | **戴维宁定理是核心**，替代定理较少单独命题 |
