## ⚡ 基本物理量与参考方向

### 1. 参考方向 (Reference Direction)
* **必要性**：电压和电流具有大小和方向双重属性。复杂电路中实际方向往往不可预知，因此必须预先设定**参考方向**。
* **属性**：参考方向是人为假定的，不影响实际物理规律。
* **判定结果**：
    * 计算结果为正 ($>0$)：参考方向与实际方向一致。
    * 计算结果为负 ($<0$)：实际方向与参考方向相反。

### 2. 欧姆定律的修订
引入参考方向后，欧姆定律的表达式需根据电压 $u$ 与电流 $i$ 的方向关系进行判定：
* **关联参考方向**（方向一致）：
  $$u = Ri$$
* **非关联参考方向**（方向相反）：
  $$u = -Ri$$
> *注：公式中的负号仅表征方向关系。*



### 3. 电位与电压
* **电位 ($v$)**：相对于参考点（零电位点）的电压。参考点通常选大地或公共接地点。
* **电压 ($u$)**：定义为两点间电位之差。例如 $a, b$ 两点间的电压为：
  $$u_{ab} = v_a - v_b$$

## 🛠 电路模型与建模思维

1. **集中参数电路 (Lumped Parameter Circuit)**：
   * **适用条件**：器件尺寸远小于工作电磁波波长。
   * **特征**：忽略分布效应，将 R、L、C 视为集总元件。
2. **分布参数模型**：
   * 适用于输电线路等长距离系统，需考虑沿线连续分布的效应。
3. **建模思维**：建模时需抓住主要矛盾、忽略次要因素。虽然研究的是理想化模型，但它们抽象自真实设备的物理本质。

## 💡 核心观念与解题思维
*  **方向意识的建立**：与以往物理课只考虑大小不同，现在的电路分析必须引入“方向”概念   。当实际方向不可直接获知时，需依赖假定的“参考方向”进行后续计算  。
*  **负号的物理意义**：计算结果或公式中的负号并不改变物理量本身的大小与本质，它相当于一个“纠错”机制，表示假定的参考方向与实际方向相反  。
*  **规范分析步骤**：电路分析中应先选定参考点（零电位点），再标定各支路电压与电流的参考方向，最后再结合计算出的功率正负来判断元件的真实性质  。

## ⚡ 参考方向与欧姆定律修订
*  **关联与非关联参考方向**：电压与电流的参考方向一致，即电流参考方向由电压参考正极流向负极（或用箭头表示时两者指向相同）   。此时欧姆定律公式为：u = i * r  。
*  **非关联参考方向**：电压与电流的参考方向相反   。若此时应用欧姆定律，需添加负号修正：u = -i * r  。
*  **电压与电动势的关系**：电动势e的实际方向由电源负极指向正极（代表电压升方向）；电压u的实际方向由正极指向负极（代表电压降方向）   。两者数值相等，但符号取决于所设的参考方向是否一致  。

## 🛠 电位与功率判别
*  **电位与电压的联系**：电位是相对于参考点的电压，参考点的电位被人为设定为0 [ e: 1, 2] 。任意两点间的电压等于这两点电位之差   。参考点选择的不同会改变各点的电位计算值，但绝对不会改变电路中任意两点之间的实际电压值  。
*  **功率计算与判定**：功率计算基于 p = u * i   。在关联参考方向下，若 p > 0，元件吸收功率（负载）；若 p < 0，元件发出功率（电源）   。在非关联参考方向下，公式需包含负号以修正方向，即 p = -u * i  。

## 💡 核心观念与电路极端状态
*  **温故知新**：再次强调了“参考方向”在功率计算中的决定性作用   。当假定元件为负载（关联参考方向）时，p = u * i > 0 表示吸收功率，p < 0 表示发出功率   ；若方向非关联，则公式变为 p = -u * i  。
* **电路极端状态**：
    *  **开路**：指电流恒为零而电压可任意，对应电路断开状态  。
    *  **短路**：指电压恒为零而电流可任意，对应两点间以理想导线连接   。二者是分析电路极端状态的基础概念  。

## ⚡ 三大基本无源元件特性
*  **电阻元件 (R)**：纯耗能的无源元件，其功率恒为正值（p = i^2 * r > 0）   。不具记忆性，任意时刻电压仅取决于该时刻电流，与历史状态无关  。
*  **电容元件 (C)**：储能、无源、有记忆的元件，其电流与电压变化率相关（ic = c * duc/dt）   。直流稳态下相当于开路（ic = 0）   。瞬时功率正负交替反映充放电过程，平均有功功率为零  。
*  **电感元件 (L)**：与电容呈对偶关系   。其电压与电流变化率相关（ul = l * dil/dt）   。直流稳态下相当于短路（ul = 0）   。同样具有记忆性，瞬时功率也正负交替，平均有功功率为零，体现能量交换而非消耗  。

## 🛠 理想独立电源模型
*  **理想电压源**：提供恒定电压，其值不随外电路变化，但输出电流由外电路决定   。短路会导致电流极大而不可允许   。实际电压源需考虑内阻，输出电压 u = us - i * rs，内阻越小越接近理想特性  。
*  **理想电流源**：提供恒定电流，其值不随外电路变化，但端电压由外电路决定   。与电压源对偶，其端电压极性需结合所接负载分析  。
*  **独立电源定义**：“独立”指其激励量（电压或电流）不受电路中其他变量控制，仅由自身参数决定  。

## 💡 核心观念与电源特性
*  理想电流源：输出电流恒定不变，不受外电路影响，但两端电压由外电路决定  。
*  操作禁忌：理想电流源绝对不允许开路。开路相当于接入无穷大电阻，会导致端电压趋于无穷大  。
*  实际电流源：由理想电流源并联一个大内阻构成，内阻越大越接近理想特性  。
*  功率计算：假定电源为负载（吸收功率）  。关联参考方向用 p = ui，非关联用 p = -ui   。若 p > 0 表示实际吸收，p < 0 表示实际发出  。

## ⚡ 基本概念与 KCL 定律
*  支路：无分支的元件串联路径  。
*  节点：三条及以上支路的连接点  。
*  回路：闭合路径  。
*  网孔：内部不含其他支路的独立回路  。
*  基尔霍夫电流定律 (KCL)：任意时刻任一节点流出电流代数和为零，即流入电流总和等于流出电流总和  。
*  广义节点：KCL 可推广至包围多个节点的封闭面（广义节点），流入该封闭面的电流总和等于流出电流总和  。

## 💡 KVL 定律与应用
*  基尔霍夫电压定律 (KVL)：核心是回路中各元件电压代数和为零  。
*  巡行方向法：列方程时需先设定巡行方向（即默认的电压正方向）  。与之同向取正、反向取负，避免混淆“电压升/降”分类带来的错误  。
*  核心推论：端口电压与路径无关。无论选择哪条路径计算两点间电压，结果恒定  。
*  支路电压计算：若支路上有多个元件（如电压源与电阻串联），须按指定正方向确定各元件电压符号后叠加  。

## ⚡ 电阻的等效化简与受控源
*  电阻串联：等效电阻等于各电阻之和（Req = R1 + R2 + ...），主要作用是分压与限流  。
*  电阻并联：等效电导等于各电导之和，等效电阻为其倒数，主要作用是分流  。
*  两电阻并联高频公式：Req = (R1 * R2) / (R1 + R2)   。分流公式为 I1 = I * R2 / (R1 + R2)  。
*  受控源电路分析：受控源分为四类（VCVS、CCVS、VCCS、CCCS），在方程中的形式与独立源相同，但需引入控制变量   。若该变量未出现在所列方程中，则需补充相应关系式  。

> **课程内容**：含受控源电路中的电源等效变换方法，与独立源变换的相同与不同之处，以及变换中不能改变受控源控制量的核心原则。

## 一、受控电源概述

### 1. 受控源概念

受控源（Controlled Source / Dependent Source）的电压或电流受电路中其他支路的电压或电流控制。

| 类型 | 符号 | 控制量 |
|:---|:---|:---|
| 受控电压源 (VCVS / CCVS) | 带受控标记的电压源 | 电压或电流 |
| 受控电流源 (VCCS / CCCS) | 带受控标记的电流源 | 电压或电流 |

### 2. 与独立电源的区别

| | 独立电源 | 受控电源 |
|:---|:---|:---|
| 电压/电流值 | 固定或已知函数 | 受其他支路控制 |
| 变换规则 | 可任意变换 | 变换时需**保证控制量不变** |

## 二、受控源的等效变换

### 1. 基本变换规则

和独立电源一样：

- **受控电压源串电阻** → **受控电流源并电阻**
- **受控电流源并电阻** → **受控电压源串电阻**

大小关系：**U = I × R**（同样适用）
方向：电流方向由**电压源负极指向正极**

### 2. ⚠️ 核心原则：控制量不能改变

这是受控源等效变换中**最重要的一点**。

> 受控源的控制量在整个电路中是**唯一确定**的，一旦变换改变了控制量所在支路的结构，控制量就不再是原来的量了，变换就错了。

**错误示例**：
- 电路中有一个受控电压源受电阻 R₁ 两端的电压 u₁ 控制
- 如果把和受控源串联的部分做了变换，导致 R₁ 被合并或改变
- 则新的电路中的 u₁ 已经≠ 原来的 u₁
- 受控源的输出电压就变成了另一个值 → **变换错误**

### 3. 什么时候可以变？

**可以**：变换的支路**不影响控制量**的计算

- 变换受控源本身（电压源↔电流源），只要控制量所在的支路没有动
- 变换独立电源，只要它和控制量无关

**不可以**：变换后控制量被破坏或改变了
- 变换后控制量所在的电阻被合并了
- 变换后控制电流 i₁ 所在的支路消失了

### 4. 变换策略

当电路中有受控源时，求某支路电流/电压时：

1. **尽量不动的支路**：你要求的那条支路
2. **可以动的支路**：与所求量无关的部分
3. **谨慎动的支路**：控制量所在支路
4. 先变换不影响控制量的部分，再逐步简化

## 三、典型例题

### 例1：含受控电压源的电路

**电路描述**：一个受控电压源（受 u₁ 控制）+ 电阻 R₁ + 独立电源混合

**解题思路**：
1. 先判断：哪个电源能变？哪个不能变？
2. 独立电压源：如果把它变换成电流源，可能会改变控制量 u₁ → ❌ 不能变
3. 受控电压源本身：可以变换成受控电流源，只要控制量 u₁ 不变
4. 逐步变换，最终化简为单一回路

### 例2：多个电源混合的电路

**电路描述**：含独立电压源 + 受控电流源 + 多个电阻

**解题思路**：
1. 观察：哪些支路是并联关系？哪些是串联关系？
2. 如果大部分是并联 → 优先转化成**电流源**形式
3. 如果大部分是串联 → 优先转化成**电压源**形式
4. 受控电流源并电阻 → 受控电压源串电阻（不改变控制电流 I）
5. 合并同类支路，逐步化简

### 例3：求特定电阻上的电流

**电路描述**：含独立电压源、受控电流源，求某电阻上的电流

**要求**：用电压源和电流源等效变换的方法

**解题思路**：
1. 把要求的支路**保留不动**（作为外部端口）
2. 站在这个端口"看进去"，对剩余部分做等效变换
3. 受控电流源→受控电压源 
4. 合并串联电阻，列出回路方程
5. 最终等效为一个电压源串一个电阻的简单电路

## 四、重要结论

### 1. 变换需谨慎

> 受控源可以像独立源一样做等效变换，但**必须保证控制量不变**。

如果变换后控制量改变了 → 这已经是**另一个电路**了，求出的结果与原电路无关。

### 2. 求什么，别动什么

| 你要求的量 | 变换建议 |
|:---|:---|
| 某条支路的电流 | **不要动这条支路**，做对外等效 |
| 某两点间的电压 | 不要动这两个点所在的支路 |
| 端口等效电阻 | 可以任意变换（端口保留） |

### 3. 等效变换 vs 列方程

| 方法 | 适合场景 |
|:---|:---|
| 电源等效变换 | 电源多、结构乱，有化简空间 |
| KCL + KVL 列方程 | 结构固定、受控源关系复杂 |
| 两者结合 | 化简一部分 + 对剩余部分列方程 |

## 五、关键总结

| 知识点 | 要点 |
|:---|:---|
| 受控源变换规则 | 和独立源一样（U = I×R，方向负→正） |
| 控制量不能变 | 变换时确保控制量所在支路未被改动 |
| 求谁的支路不动 | 求哪个电流/电压，尽量不改变它的支路 |
| 对外等效 | 站在所求端口"看进去"，对外部做等效 |
| 化简策略 | 并联多→变电流源，串联多→变电压源 |
