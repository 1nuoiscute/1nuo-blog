<h2>Lesson 9 网孔电流法与回路电流法</h2>

> **课程内容**：本课讲解网孔电流法的标准形式与特殊处理（电流源、受控源），从网孔法到回路电流法的推广，以及图论基础（树、树枝、连枝、基本回路、基本割集）。

---

## 一、网孔电流法回顾

### 1. 基本思路

- 以**网孔电流**为变量
- 对每个网孔列写 **KVL 方程**（电压方程）
- 方程个数 = 网孔个数

### 2. 标准形式

对于有三个网孔的电路：

$$R_{11}I_1 + R_{12}I_2 + R_{13}I_3 = U_{s1}$$
$$R_{21}I_1 + R_{22}I_2 + R_{23}I_3 = U_{s2}$$
$$R_{31}I_1 + R_{32}I_2 + R_{33}I_3 = U_{s3}$$

其中：
- $R_{kk}$ = 网孔 k 中所有电阻之和（自阻，总是正）
- $R_{kj}$ = 网孔 k 和网孔 j 的公共电阻（互阻，符号取决于方向）
- $U_{sk}$ = 网孔 k 中电压源电压的代数和

---

## 二、特殊情况的处理

### 1. 含有独立电流源

**方法一（通用）**：假设电流源两端电压为未知量
- 多了一个未知量 → 多补一个方程（电流源电流 = 相关网孔电流的代数和）

**方法二（推荐）**：选择合适的回路，让电流源支路只属于一个回路
- 该回路的电流 = 电流源电流（已知）
- 少列一个方程，少一个未知量
- 需要选择**广义回路**（不一定是网孔，可能是外圈大回路）

> 本质上这就从"网孔电流法"推广到了"回路电流法"。

### 2. 含有受控源

- 先把受控源**当作独立源**处理，列方程
- 再补充**控制变量的方程**

**步骤**：
1. 列网孔 KVL 方程（受控源当成独立源）
2. 补充受控源的控制量方程
3. 联立求解

### 3. 超网孔（Supermesh）

当电流源跨在两个网孔之间时：
- 把这两个网孔连同中间的电流源看作一个"超网孔"
- 对超网孔列 KVL 方程
- 补充电流源电流与网孔电流的关系方程
- 优点：少列一个方程

---

## 三、从网孔法到回路法

### 1. 网孔电流法的局限性

- **仅适用于平面电路**（可以画在平面上无交叉）
- 立体电路、复杂电路无法使用网孔
- 当出现多个独立电流源时，难以选择合适的回路

### 2. 回路电流法

回路电流法是网孔法的**推广与一般化**：
- 不再局限于"网孔"，而是选择**独立回路**
- 需要一套理论来系统性地找到所有独立回路
- 网孔电流法只是回路电流法在平面电路中的一个特例

## 四、图论基础

### 1. 基本概念

| 概念 | 定义 |
|:---|:---|
| 图（Graph） | 由节点和支路组成的拓扑结构，不考虑元件类型 |
| 子图 | 原图的某一部分（节点+支路） |
| 路径 | 从一个节点到另一个节点经过的支路序列 |
| 连通图 | 任意两节点之间至少存在一条路径 |
| 有向图 | 支路有参考方向的图 |
| 孤立节点 | 没有支路连接的节点 |

**移走支路 ≠ 移走节点**（节点可独立存在）
**移走节点 = 移走该节点连接的所有支路**

### 2. 树（Tree）

树是图论中最重要的概念之一。一棵树需满足三个条件：

1. **是连通图**（不能分成两块）
2. **包含原图所有节点**
3. **不含回路**

> 🌲 **形象记忆**：像一棵真实的树——树干树枝发散出去，不会绕回来，也不会断开。

**性质**：
- 树是连接全部节点所需的**最少支路集合**
- 一个有 n 个节点的图，树枝数 = **n − 1**
- 一个电路可以选择不同的树

### 3. 树枝与连支

| 名称 | 定义 | 数量 |
|:---|:---|:---|
| **树枝**（Tree Branch） | 组成树的支路 | n − 1 |
| **连支**（Link） | 不属于树的支路 | b − (n − 1) |

b = 总支路数，n = 总节点数

### 4. 基本回路（Fundamental Loop / Basic Loop）

- 定义：**仅含一条连支** + 若干树枝构成的回路
- 每个基本回路中的连支是该回路的**独有支路**（其他回路没有）
- 因此这些回路方程**独立**

**基本回路的个数 = 连支的个数 = b − n + 1**

**方向**：基本回路的方向通常取为连支的方向（省事）

### 5. 基本割集（Fundamental Cut-set）

- 对应概念：基本回路靠"连支"，基本割集靠"**树枝**"
- 定义：**仅含一条树枝** + 若干连支构成的割集
- 一个基本割集对应一个**广义节点**（闭合面）

**基本割集的个数 = 树枝的个数 = n − 1**

**判断方法**：做一个高斯闭合面（封闭面），切割到的支路集合就是一个割集。如果这个割集里**只有一条树枝**，就是基本割集。

---

## 五、回路电流法的完整步骤

### 1. 基本步骤

1. **画拓扑图**（去掉元件类型，只留节点和支路）
2. **选树**（注意策略，见下方）
3. 确定连支 → 每个连支对应一个基本回路
4. 设定回路电流（方向 = 连支方向）
5. 列写基本回路的 KVL 方程
6. 求解回路电流
7. 由回路电流求支路电流或其所需量

### 2. 选树的策略

| 情况 | 选树建议 |
|:---|:---|
| 含有独立**电流源** | 电流源选为**连支**（回路电流已知，不用列方程） |
| 含有独立**电压源** | 电压源选为**树枝** |
| 含有**受控源** | 受控源先当独立源处理 |
| 受控源的**控制量** | 尽量把控制量选为**连支**（少了一个方程） |

> 💡 电流源 → 连支（已知量，少列方程）
> 
> 电压源 → 树枝（参考量，便于列方程）

### 3. 与网孔电流法的关系

- 如果选树时选择了**中间呈十字形**的树枝，基本回路就是**网孔**
- **网孔电流法是回路电流法的一个特例**
- 回路电流法适用于**平面和立体电路**

---

## 六、例题分析

### 例1：常规电路（无特殊元件）

选最简单的树（中间十字）→ 基本回路 = 网孔 → 列标准形式方程

### 例2：含独立电流源

- 电流源选为连支 → 该回路电流 = 电流源电流（已知）
- 只需对其他回路列方程
- 再补充电流源与回路电流的关系方程

### 例3：含受控源

- 先当独立源列方程
- 再补充受控源的控制变量方程

### 例4：电流源并联电阻的化简

- 如果电流源并联了电阻 → 可等效变换为电压源串电阻
- 减少方程个数
- 注意：**求电流的支路尽量不动**

---

## 七、关键总结

| 概念 | 要点 |
|:---|:---|
| 网孔电流法 | 以网孔电流为变量，列KVL方程 |
| 特殊元件 | 电流源→假定电压+补方程 / 超网孔 |
| 受控源 | 先当独立源，再补控制方程 |
| 图论 | 节点+支路，去掉元件类型的拓扑 |
| **树** | 连通、含所有节点、无回路，树枝=n−1 |
| 连支 | 非树支路，数量=b−n+1 |
| 基本回路 | 1条连支+树枝，回路数=连支数 |
| 基本割集 | 1条树枝+连支，割集数=树枝数 |
| 回路电流法 | 网孔法的推广，适用于平面+立体 |
| 选树策略 | 电流源→连支，电压源→树枝 |

<h2>Lesson 10 回路电流法应用实例</h2>

> **课程内容**：含多个独立电流源的电路运用回路电流法求解，重点练习选树策略与列方程。

---

## 一、为什么用回路电流法而不是网孔法？

| 情况 | 推荐方法 |
|:---|:---|
| 常规电路（无电流源） | 网孔电流法（更简单，不用画树） |
| 含 1 个独立电流源 | 网孔法/回路法都可以（超网孔或选树） |
| **含多个独立电流源** | **回路电流法 + 选树策略**（更系统） |
| 含受控源 | 回路法 + 补控制方程 |

---

## 二、典型例题：含两个独立电流源

### 例题描述

电路中有两个独立电流源（可能包含受控源），要求求某条支路的电流 i₁。

### 解题步骤

**1. 确定节点数，选树**

- 电路有 5 个节点
- 树枝数 = n − 1 = 4
- 策略：**电流源选为连支**
  - 两个电流源所在的支路**不能当树枝**
  - 从剩余支路中选出 4 条构成一棵树
  - 树必须满足：连通、含全部节点、无回路

**2. 老师选的树**

四条树枝构成树，两个电流源支路作为连支：
- 树支路从外围走，避开电流源
- 确保覆盖所有 5 个节点
- 没有形成闭合回路

**3. 确定基本回路**

每条连支 + 若干树枝 = 一个基本回路
- 连支① → 基本回路 1（回路电流 = 电流源电流，已知）
- 连支② → 基本回路 2（回路电流 = 电流源电流，已知）
- 剩余连支 → 基本回路 3、4...（需列 KVL 方程求解）

**4. 列方程**

- 电流源所在的回路 → **不需要列方程**（电流已知）
- 其他基本回路 → 列 KVL 方程
- 如果还有**受控源** → 再补充控制变量的方程

**5. 求解**

联立方程求得回路电流 → 由回路电流求支路电流 i₁

---

## 三、选树的练习技巧

| 节点数 | 树枝数 | 策略 |
|:---|:---|:---|
| 3 | 2 | 避开电流源，选两条不形成回路的支路 |
| 4 | 3 | 像一棵树一样发散出去 |
| 5 | 4 | 从外围绕，或从中间发散 |
| n | n−1 | 树连接所有节点且无回路 |

**要点**：
1. 先标出所有**节点**
2. 把电流源支路**标记出来**（选为连支）
3. 从剩下的支路中选 n−1 条构成一棵树
4. 检查：树是否连通？是否覆盖所有节点？是否有回路？

> 💡 选树不唯一！做作业时发现和答案选的树不同很正常，结果应该一样。

---

## 四、回路电流法完整流程总结

```
1. 确定节点数 n，总支路数 b
2. 选树（避开电流源选连支，避开电压源选树枝）
3. 树枝数 = n−1，连支数 = b−(n−1)
4. 每条连支对应一个基本回路
5. 电流源所在回路 → 电流已知，不列方程
6. 其他回路 → 列 KVL 方程
7. 有受控源 → 补控制方程
8. 求解 → 由回路电流求支路电流
```

---

## 💡 关键总结

| 要点 | 说明 |
|:---|:---|
| 多个电流源 | 回路法比网孔法更方便，选树即可系统处理 |
| 电流源→连支 | 回路电流已知，少列方程 |
| 选树不唯一 | 不同树→不同回路→方程不同→结果相同 |
| 无电流源 | 直接网孔法更快，不用画树 |
| 有受控源 | 当独立源处理，再补控制方程 |
