<h2>Lesson 18 相位差与初相位的概念解析</h2>

> **课程内容**：本课讲解正弦交流电中有效值与最大值的关系、初相位与相位差的概念，以及用复数相量表示正弦量的方法及其在电路分析中的应用。

---

## 一、正弦交流电基础回顾

### 1. 正弦量的三要素

正弦电流函数：**i = I_m sin(ωt + φ)**

| 要素 | 符号 | 含义 |
|:---|:---|:---|
| 幅值（最大值） | I_m / U_m | 波形峰值 |
| 角频率 | ω = 2πf | 变化快慢 |
| 初相位 | φ | t=0 时的相位角 |

### 2. 最大值与有效值

**I = I_m / √2 , U = U_m / √2**

- **有效值**（I、U，大写无下标）：体现热效应，用于功率和能量计算
- **最大值**（I_m、U_m，下标 m）：用于瞬时值分析
- 电路计算通常采用**有效值**

---

## 二、初相位与相位差

### 1. 初相位

- **定义**：t=0 时刻正弦量的相位角 φ
- 反映波形的**起始位置**
- 取值范围：**主值区间** （-180° 到 +180°]，即（-π 到 +π]
  - 超过 +180° 需减去 360°（例如：240° → -120°）
  - 低于 -180° 需加上 360°

### 2. 相位差

**φ = φ_u − φ_i**

- **定义**：**同频率**正弦量之间初相位之差
- **特性**：**与时间 t 无关**，为恒定值！不随你开关切入的时刻而改变
- 必须在**主值区间**内表示

### 3. φ 的三种情况

| φ 取值 | 关系描述 | 波形表现 |
|:---|:---|:---|
| φ > 0 | **电压超前于电流**（电流滞后于电压） | 电压波峰先到达 |
| φ < 0 | **电流超前于电压**（电压滞后于电流） | 电流波峰先到达 |
| φ = 0 | **同相位** | 同时到达峰值 |
| φ = ±180° | **反相位** | 一正一负，方向相反 |

---

## 三、相位比较的前提条件

两个正弦量进行相位比较时，必须满足**三同原则**：

1. **同频率** ❗ 最重要
   - 不同频率的正弦量之间**不存在恒定相位差**
   - 异频量**不可直接比较相位**或进行相量运算

2. **同函数形式**（同为 sin 或同为 cos）
   - sin 与 cos 可通过 ±90° 相位转换统一
   - 本教材采用 cos 函数为基准

3. **同符号**（幅值均为正值）
   - 若出现负号（如 −A·cos(ωt+φ)），需通过 ±180° 调整为正幅值

4. 计算结果若超出主值区间，需**归化**至（−180°, +180°]

---

## 四、正弦量的相量表示

### 1. 为什么可以用相量？

**核心观察**：同频率正弦量在电路中进行**加减、微分、积分**运算后，**频率保持不变**。

- 电源（激励）的频率 ω 一经确定，电路中所有响应（电压、电流）的频率都与电源相同
- 因此 ω 可作为已知常量而无需重复求解
- **只需关注幅值（或有效值）和初相位两个要素**

### 2. 欧拉公式与相量

$$I_m e^{j(\omega t + \varphi)} = I_m\cos(\omega t + \varphi) + jI_m\sin(\omega t + \varphi)$$

- 实部 → cos 函数
- 虚部 → sin 函数

提取不含 t 的部分：

$$\dot{I} = I_m e^{j\varphi} = I_m \angle \varphi$$

- **İ 称为电流的相量**
- 模 = 幅值（或有效值）
- 辐角 = 初相位

### 3. 相量与正弦量的关系

$$i(t) = \Re[\dot{I} e^{j\omega t}]$$

> ⚠️ **重要区分**：
> - 相量 İ 是**复常数**（不含 t）
> - 正弦量 i(t) 是时间的函数
> - 二者**只有对应关系（表示），没有相等关系**
> - 相量法只是分析求解正弦交流电路稳态响应的**数学工具**

### 4. 最大值相量与有效值相量

- **最大值相量**：İ_m = I_m ∠ φ（模为最大值）
- **有效值相量**：İ = I ∠ φ（模为有效值）

电路中通常使用**有效值相量**。

### 5. 相量的三种表示形式

- **极坐标形式**：U̇ = U ∠ φ
- **指数形式**：U̇ = U e^(jφ)
- **代数形式**：U̇ = a + jb

---

## 五、相量图

### 1. 画法

- 各正弦量以**有向线段**表示
- **长度**代表幅值（或有效值）
- 与正实轴的**夹角**代表初相位
- 两相量之间的**夹角**即为其相位差

### 2. 超前与滞后

- 逆时针方向旋转，角度增大
- **角度大者 → 超前**
- **角度小者 → 滞后**
- 图形直观反映动态相位关系

### 3. 参考相量

- 选取某一电压或电流的初相位设为 0°（称为**参考相量**）
- 其余量依据其与参考量的相位差确定相对位置
- **不影响各量间固有的相位关系**
- 比喻：以一个人为坐标原点，大家都跟他比身高差，他站地上最方便

## 六、书写规范与符号区分

| 符号样式 | 含义 | 示例 |
|:---|:---|:---|
| 小写字母 | **瞬时值** | i、u |
| 大写字母（无点） | **有效值**（标量，无相位信息） | I、U |
| 大写字母 + 下标 m | **最大值** | I_m、U_m |
| 大写字母 + 上点 · | **相量**（含幅值+相位） | İ、Ü |

> 🔑 **关键记忆**：
> - 正弦量（含 t） ≠ 相量（不含 t）
> - 相量必须打点，无点 = 有效值标量
> - i（瞬时值）≠ I（有效值）≠ İ（相量）
> - 电阻 R 不是正弦量，所以 R 不需要打点

---

## 七、课堂例题

**例1**：判断 i₁ = 5√2·cos(ωt+45°) 与 i₂ = 10·cos(ωt+30°) 的相位关系
- ✅ 同频率 ω、同函数 cos、同符号（正）
- 相位差 φ = 45° − 30° = 15° > 0
- → i₁ **超前** i₂ 一个 15° 角

**例2**：u = 10·cos(ωt+60°) 与 i = 5·sin(ωt−15°)
- ⚠️ 函数形式不同！sin 和 cos 不能直接相减，需先统一

**例3**：u = 10·cos(ωt) 与 i = 5·cos(2ωt)
- ❌ 频率不同，**不可**进行相位比较

**例4**：u = −10·cos(ωt+30°) 与 i = 5·cos(ωt+60°)
- ⚠️ 先把负号转为 ±180° 再比较

---

## 八、关键总结

| 知识点 | 要点 |
|:---|:---|
| 有效值 ↔ 最大值 | I = I_m/√2，计算用有效值 |
| 相位差 | 同频率正弦量初相位之差，**恒定值** |
| 比较前提 | **同频率、同函数、同符号** + 主值区间 |
| 相量 | 复常数，不含 t，**表示**不等于**相等** |
| 相量图 | 长度=幅值，夹角=相位差 |
| 符号规则 | 瞬时小写，有效值大写（无点），相量大写加点 |

<h2>Lesson 20 交流电路中的向量欧姆定律</h2>

> **课程内容**：本课讲解交流电路中的阻抗与导纳概念、向量形式的欧姆定律、RLC串并联电路分析，以及通过习题巩固节点电压法和回路电流法在正弦稳态电路中的应用。
> **强调**：交流电路分析方法与直流电路相同，唯一区别是将电阻 R 换为阻抗 Z，电压电流变为向量形式，运算本质是复数运算。

---

## 一、交流电路分析方法总览

### 核心思想

交流电路的分析方法与直流电路**完全一致**，贯彻整个课程：
1. 电路基本定律（基尔霍夫定律）
2. 节点电压法
3. 回路电流法（网孔电流法）
4. 叠加定理、替代定理等
5. 戴维南等效、诺顿等效

**唯一不同**：
- 电阻 R → **阻抗 Z**（复数）
- 电压、电流 → **向量形式**（带点）
- 运算法则 → **复数运算**

> 🔑 **关键理解**：学会把交流电路"翻译"成直流的形式，把 Z 看作 R 的升级版。

---

## 二、阻抗 Z 的概念

### 1. 阻抗的定义

$$Z = \frac{U}{I} \quad \text{（向量形式）}$$

- 满足向量形式的欧姆定律：**U̇ = Z · İ**
- Z 是**复数**，包含实部和虚部

### 2. 阻抗的组成

$$Z = R + jX$$

| 部分 | 符号 | 含义 | 典型值 |
|:---|:---|:---|:---|
| 实部（电阻） | R | 电阻分量 | R（实数） |
| 虚部（电抗） | X | 电抗分量 | XL = ωL（感抗）或 XC = −1/ωC（容抗） |
| 模值 | \|Z\| | 阻抗大小 | \|Z\| = √(R² + X²) |
| 辐角 | φ | 阻抗角 | φ = arctan(X/R) |

### 3. 三个元件的向量形式欧姆定律

| 元件 | 向量形式 | 阻抗表达式 |
|:---|:---|:---|
| 电阻 R | U̇_R = R · İ | Z_R = R |
| 电感 L | U̇_L = jωL · İ = jX_L · İ | Z_L = jωL = jX_L |
| 电容 C | U̇_C = −j(1/ωC) · İ = −jX_C · İ | Z_C = −j/ωC = −jX_C |

- **感抗** X_L = ωL = 2πfL（正虚部）
- **容抗** X_C = 1/ωC = 1/(2πfC)（负虚部）
- **j 的意义**：代表 90° 相位旋转（电感超前，电容滞后）

### 4. 阻抗三角形

```
        /|
  |Z|  /  |
      /    | X (虚部)
    / φ    |
   /_______|
      R (实部)
```

- 横轴 = R（实部）
- 纵轴 = X（虚部）
- 斜边 = |Z|（模值）
- 角度 φ = 阻抗角 = 电压超前电流的角度

---

## 三、导纳 Y

### 1. 定义

$$Y = \frac{1}{Z} = \frac{İ}{U̇}$$

- 单位：**西门子（S）**
- 导纳 Y 是阻抗 Z 的**倒数**

### 2. 组成

$$Y = G + jB$$

| 部分 | 符号 | 含义 |
|:---|:---|:---|
| 电导 | G | 实部，G = R/(R²+X²) |
| 电纳 | B | 虚部，B = −X/(R²+X²) |
| 模值 | \|Y\| | 1/\|Z\| |

### 3. 使用说明

- 本学期（电路分析）**导纳使用不多**
- 下学期**二端口网络**会用到导纳
- **电力系统分析**中大量使用导纳（节点电压法中的系数矩阵即为导纳矩阵）
- 对同学们而言，直接用 Z 进行计算通常更直观

---

## 四、RLC 串联与并联

### 1. 串联（电流相同）

$$U̇ = U̇_R + U̇_L + U̇_C = Z · İ$$

$$Z = R + j(X_L - X_C) = R + j(ωL - \frac{1}{ωC})$$

**注意**：如果 X_L > X_C，电路呈**感性**（电压超前电流）；反之呈**容性**。

### 2. 并联（电压相同）

$$İ = İ_R + İ_L + İ_C = \frac{U̇}{Z_R} + \frac{U̇}{Z_L} + \frac{U̇}{Z_C}$$

**分流公式**（与电阻并联类似，R → Z）：

$$İ_1 = \frac{Z_2}{Z_1 + Z_2} · İ \quad \text{（两个阻抗并联）}$$

> ⚠️ 所有运算均为**复数运算**，不能直接代有效值！

---

## 五、参考相量的选择

### 1. 核心原则

| 原则 | 说明 |
|:---|:---|
| 选择**已知量** | 已知电压则选电压为参考，已知电流则选电流为参考 |
| 设角度为 0° | 通常设参考相量的初相位为 0°，简化计算 |
| 相对关系不变 | 所有量之间的相位差与参考选择无关 |

### 2. 为什么可以任意选？

- 参考相量相当于坐标系的"零点"
- 所有量之间的**相对位置关系不变**
- 比喻：以一个人为原点，大家都跟他比身高差

### 3. 常见选择策略

| 电路情况 | 推荐参考相量 |
|:---|:---|
| 串联电路 | 选电流 İ 为参考（0°），因为各元件电流相同 |
| 并联电路 | 选电压 U̇ 为参考（0°），因为各支路电压相同 |
| 已知端口电压 | 选 U̇ 为参考 |
| 已知端口电流 | 选 İ 为参考 |

---

## 六、解题步骤（标准流程）

### Step 1：设定参考相量
- 选择已知电压或电流为参考，角度设为 0°

### Step 2：计算总阻抗 Z
- 串联：Z = Z₁ + Z₂ + ...
- 并联：Z = (Z₁⁻¹ + Z₂⁻¹ + ...)⁻¹

### Step 3：用欧姆定律求未知量
- İ = U̇ / Z（已知电压求电流）
- U̇ = Z · İ（已知电流求电压）

### Step 4：求各元件电压/电流
- 串联电路中，各元件电流相同
- 并联电路中，各元件电压相同

### Step 5：根据需要转换回瞬时值（正弦量）
- İ → i(t) = I_m · cos(ωt + φ_i)
- U̇ → u(t) = U_m · cos(ωt + φ_u)
- 有效值 → 最大值：I_m = √2 · I

## 七、阻抗三角形与电压三角形

### 1. 阻抗三角形

```
        /|
  |Z|  /  |
      /    | X
    / φ    |
   /_______|
      R
```

### 2. 电压三角形

对于串联 RLC 电路：

```
        /|
  U    /  |
      /    | U_L − U_C
    / φ    |
   /_______|
      U_R
```

### 3. 功率因数角

$$\cos φ = \frac{R}{|Z|} = \frac{U_R}{U}$$

- φ 同时扮演多个角色：
  - **阻抗角**（Z 的辐角）
  - **电压电流差角**（φ_u − φ_i）
  - **功率因数角**

> 💡 电力系统中，系统通常偏**感性**（φ > 0，电压超前电流）

---

## 八、电路谐振

### 1. 谐振条件

当 X_L = X_C 时，电路发生谐振：

$$ωL = \frac{1}{ωC}$$

### 2. 谐振特征

| 特性 | 说明 |
|:---|:---|
| 电抗抵消 | X_L − X_C = 0 |
| 对外呈阻性 | Z = R（纯实数） |
| 电压电流同相 | φ = 0°，cos φ = 1 |
| 电流最大（串联） | İ = U̇ / R |

---

## 九、节点电压法与回路电流法（交流版）

### 1. 方法迁移

直流电路的方法**全部适用**，只需：
- 电阻 R → 阻抗 Z
- 电流/电压 → 向量形式（带点）

### 2. 节点电压法要点

$$ \begin{bmatrix} Y_{11} & Y_{12} & \cdots \\ Y_{21} & Y_{22} & \cdots \\ \vdots & \vdots & \ddots \end{bmatrix} \begin{bmatrix} U̇_1 \\ U̇_2 \\ \vdots \end{bmatrix} = \begin{bmatrix} İ_{s1} \\ İ_{s2} \\ \vdots \end{bmatrix} $$

- 自导纳 = 与该节点相连的各支路导纳之和
- 互导纳 = 两节点间支路导纳的负值
- 电流源串联阻抗 → **电流源视为理想源**，串联阻抗不影响其电流

### 3. 回路电流法要点

- 选择树（树枝和连枝）
- 以回路电流为变量列 KVL 方程
- **电流源**是列电压方程时的特殊元件（两端电压未知）
- 处理方式：
  1. 尽可能让电流源所在支路为连枝
  2. 增加补充方程

## 十、课堂例题要点

### 例1：串联 RLC 求电流和各元件电压

步骤：
1. U̇ = 100∠0°（设参考）
2. Z = R + j(X_L − X_C)
3. İ = U̇ / Z = 4∠−53.13° A
4. U̇_R = R · İ, U̇_L = jX_L · İ, U̇_C = −jX_C · İ
5. 若要求 i(t)：i = 4√2 · cos(ωt − 53.13°)

> ⚠️ **易错**：U = U_R + U_L + U_C（向量和）≠ U_R + U_L + U_C（代数和）！
> 取模值时才能直接代数相加：U = √(U_R² + (U_L − U_C)²)

### 例2：选定参考相量的灵活性

- 若设电流为参考（0°），则 U̇_R 与 İ 同向，U̇_L 超前 90°，合成得 U̇
- 整体旋转不影响各量间相对位置

### 例3：并联电路分流（思路同直流）

- 两阻抗 Z₁、Z₂ 并联 → 分流公式：İ₁ = Z₂/(Z₁+Z₂) · İ
- 只是电阻换成了复数阻抗

### 例4：谐振时电流最小问题

- 两并联支路：一个 RL 感性支路 + 一个纯电容支路
- 当电容电流的虚部恰好抵消感性支路电流的虚部时，总电流仅剩实部
- 此时 I 最小，电路对外呈阻性（谐振）

## 十二、关键总结

| 知识点 | 要点 |
|:---|:---|
| 交流分析本质 | 直流方法 + 复数运算，R → Z |
| 阻抗 Z | Z = R + jX，欧姆定律 U̇ = Z·İ（向量形式） |
| 导纳 Y | Y = 1/Z = G + jB，单位西门子（S） |
| 参考相量 | 选已知量设为 0°，相对关系不变 |
| 复数运算 | 实部与实部相等，虚部与虚部相等 |
| 有效值 ≠ 向量和 | 有效值直接相加错误！要用复数运算 |
| 谐振 | X_L = X_C，电路呈阻性，电流最大（串联） |
